#  Frame Zone  
以攝影為核心的一站式智慧整合平台

Frame Zone 是一個整合 **AI 照片分類、社群互動、購物中心、攝影師預約與影音平台** 的全端專題作品。  
本專案以實務導向為核心，驗證團隊在 **前後端整合、雲端服務、即時系統與 AI 應用** 上的開發能力。

---

##  專案簡介

在攝影相關平台中，使用者往往需要在多個系統之間切換，  
例如照片管理、社群分享、商品購買、服務預約與影音學習。

Frame Zone 以「攝影」作為核心，將上述需求整合於單一平台中，  
打造一個從 **創作 → 管理 → 交流 → 商業變現** 的完整攝影生態圈。

---

##  專案目標與核心價值

本專題採用「**模組化分工設計**」，五大核心系統彼此獨立、相互整合，  
讓每位成員能完整負責一個可落地的產品模組，並實際解決真實使用情境中的問題。

---

### 1. AI 照片分類與會員系統  
**（身份驗證、會員中心、照片管理）**

- 實作 **JWT 身份驗證** 與權限控管流程  
- 支援 **Google OAuth 第三方登入（SSO）**  
- 設計安全的密碼重置機制（一次性 Token + **bcrypt** 雜湊）  
- 整合 **Azure Computer Vision、Google Maps API、Azure OpenAI**  
- 將 AI 分析結果轉換為可搜尋、可分類的智慧標籤系統  

---

### 2. 社群互動與即時通訊系統  
**（貼文、留言、追蹤、聊天室）**

- 實作貼文與 **多層級留言結構**  
- 後端雙重權限驗證，確保僅本人可編輯 / 刪除  
- 追蹤 / 取消追蹤即時同步，影響內容顯示結果  
- 使用 **SignalR** 實作即時聊天室，支援跨裝置同步  
- 前端採 **RWD 響應式設計**，提升使用體驗  

---

### 3. 攝影商品購物中心系統  
**（商品瀏覽、購物流程、金流串接）**

- 商品搜尋、分類、價格排序與關鍵字過濾  
- 使用 **記憶體快取** 處理熱門商品點擊數，降低資料庫負擔  
- 透過 **LocalStorage** 實作近期瀏覽紀錄  
- 使用 **Regex** 清洗商品名稱，實作類似商品推薦  
- 串接 **ECPay 綠界金流**，完成實際付款流程  
- 設計完整訂單、地址與評價機制  

---

### 4. 攝影師／工作室預約系統  
**（搜尋、預約、狀態管理）**

- 複合式搜尋（地區、日期、拍攝類型）  
- 標籤 **聯集 / 交集演算法**，確保搜尋結果精準  
- 使用 **RxJS** 實作前端全域狀態同步  
- 攝影師方案與預約卡即時連動  
- 預約狀態視覺化，方便使用者追蹤流程  

---

### 5. 影音創作與內容平台  
**（影片上傳、審核、轉碼、播放）**

- 使用 **GUID** 管理影片檔案，避免命名衝突  
- 透過 **FFmpeg** 解析影片資訊並產生縮圖  
- 串接 **Sightengine AI** 進行內容安全審核  
- 非同步轉碼為 **HLS 串流（m3u8 / ts）**  
- 使用 **HLS.js** 實作多畫質播放器  
- 提供曝光輔助、顏色模式與成效數據分析  

---

##  技術架構

- **Frontend**：Angular、RxJS、RWD  
- **Backend**：ASP.NET Core Web API（三層式架構）  
- **Database**：SQL Server  
- **Realtime**：SignalR  
- **Cloud / AI**：Azure Blob Storage、Azure AI、Azure OpenAI  
- **Third Party**：Google OAuth、Google Maps API、ECPay、Sightengine  
- **Version Control**：Git、GitHub
